# 📊 Snapdeal Product Analysis Dashboard (Power BI Internship Project)

## 📌 Project Overview

This project was developed as part of my **Data Analytics Internship**, focusing on analyzing product data from Snapdeal (an e-commerce platform). The goal was to extract meaningful insights from raw product data and present them through interactive dashboards using **Power BI**.

---

## 🎯 Objectives

* Analyze product pricing and discount trends
* Understand customer behavior using ratings and reviews
* Identify top-performing categories and brands
* Build interactive dashboards for business insights

---

## 🛠️ Tools & Technologies Used

* **Power BI** – Data visualization & dashboard creation
* **Microsoft Excel / CSV** – Data storage
* **Python (optional)** – Data cleaning & preprocessing
* **Web Scraping** – Data collection
